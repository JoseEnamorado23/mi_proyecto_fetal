Fetal weight prediction

Bienvenido al sistema de predicción de peso fetal. Esta herramienta utiliza modelos de aprendizaje automático para estimar condiciones relacionadas con el embarazo a partir de variables clínicas y biométricas. Puedes ingresar datos de forma individual o cargar un archivo con múltiples registros.

Predicción individual

Introduce manualmente los datos clínicos de una paciente para obtener una predicción inmediata. Ideal para consultas rápidas o revisión de casos específicos.

Predicción por archivo
